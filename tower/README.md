# The Magician's Tower

This directory contains development notes for golem itself — the harness,
the cartridge system, and the conventions that hold it all together.

**WARNING**: If you are a model operating a golem session (i.e. you were
bootstrapped via kernel.md and are processing a task), do NOT read this
directory. It is not for you. It will confuse your instructions. This is
for humans and models who are developing golem, not using it.

Contents:

- `cart-development.md` — how to develop cartridges
- `in-the-beginning.md` — origin story and design decisions
